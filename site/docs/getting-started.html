Getting Started

Requirements

Claude Code CLI installed. That’s it.

Install

From the marketplace

claude install @eaven/gigo

Manual install

git clone https://github.com/Eaven/gigo.git
cp -r gigo/skills/ ~/.claude/skills/

First Run

Open any project and say “gigo.”

Here’s what happens:

  1. GIGO checks your project. It looks for an existing CLAUDE.md and .claude/ directory. If nothing exists, it starts a conversational kickoff.
  2. You describe what you’re building. Be as vague or specific as you want — “a Rails API for library reservations” or “kids books idk.”
  3. GIGO researches authorities. It finds the best practitioners in the domain, identifies their specific philosophies, and proposes blended personas — 2-3+ real practitioners per persona.
  4. You react and refine. “Drop that one.” “Add a testing expert.” “Who would they disagree with?” The skill adjusts.
  5. Lock it in. GIGO writes your project files — a complete expert team, ready to work.

What Gets Created

FileWhat it doesToken cost
CLAUDE.md Team roster — personas with blended authorities, autonomy model, quick reference Auto-loaded every conversation
.claude/rules/ Standards, workflow, domain extensions. ~60 line cap per file (research-informed) Auto-loaded every conversation
.claude/references/ Deep knowledge — authority profiles, technique catalogs, domain deep-dives Zero cost when unused. Loaded on demand via “When to Go Deeper” pointers

The two-tier architecture keeps always-on context lean and loads depth only when relevant. This is based on Gloaguen et al. (2026), which found that bloated context reduces task success rates while increasing cost by 20%+.

Next Steps

SkillWhen to use it
gigo:plan Turn your first task into a spec and implementation plan. Brainstorms first, then structures.
gigo:execute Run the plan with agent teams or subagents. Workers get the spec, not the personas.
gigo:review Two-stage code review — spec compliance first, then engineering quality.
gigo:snap Run at session end to audit rules and protect the project from context bloat.
